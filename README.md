# ResBnb App

ResBnb App with Ionic, Angular

### Installation & Development

1. clone this repo: `git clone https://github.com/fasfalisn/resbnb.git`
2. `cd resbnb`
3. `npm install`
4. run `ionic serve` from a terminal
